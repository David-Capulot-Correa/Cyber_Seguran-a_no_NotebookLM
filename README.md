# 🛡️ Caderno Temático - Técnicas de Evasão em Cibersegurança com NotebookLM

## 📚 Contexto

Este projeto foi desenvolvido como parte do desafio da DIO utilizando o NotebookLM como ferramenta de estudo baseada em Inteligência Artificial.

O tema escolhido foi **Técnicas de Evasão em Cibersegurança**, com foco em compreender como ameaças modernas tentam evitar mecanismos de detecção e quais estratégias podem ser utilizadas para identificar e mitigar esses comportamentos.

O objetivo é desenvolver uma visão crítica sobre técnicas de evasão utilizadas por agentes maliciosos e entender como profissionais de segurança podem fortalecer a defesa dos ambientes computacionais.

---

# 🎯 Objetivos

* Compreender o conceito de evasão em cibersegurança.

* Estudar boas práticas de monitoramento e resposta a incidentes.

# ⚠️ Cicatrizes (Troubleshooting)

### Problema

Algumas respostas eram muito superficiais.

**Solução**

Adicionar contexto ao prompt.

Exemplo:

> Explique considerando o framework MITRE ATT&CK e apresente exemplos defensivos.

---

### Problema

Excesso de informações.

**Solução**

Solicitar limite de tópicos.

Exemplo:

> Resuma em até 8 tópicos.

---

### Problema

Pouca organização.

**Solução**

Pedir respostas em tabelas, listas ou mapas mentais.

# 📘 Miniguia de Estudos

## O que é Evasão?

Evasão é o conjunto de técnicas utilizadas por ameaças para reduzir as chances de serem detectadas por mecanismos de segurança durante suas atividades em um ambiente computacional.

O estudo dessas técnicas permite desenvolver mecanismos de defesa mais eficientes.

---

## Principais Conceitos

* Defesa em profundidade
* Monitoramento contínuo
* Inteligência de ameaças
* Indicadores de comprometimento (IoCs)
* Indicadores de ataque (IoAs)
* Detecção baseada em comportamento
* Framework MITRE ATT&CK

---

## Boas Práticas de Defesa

* Atualizar sistemas regularmente.
* Implementar autenticação multifator.
* Utilizar soluções EDR e SIEM.
* Monitorar logs continuamente.
* Aplicar o princípio do menor privilégio.
* Realizar treinamentos de conscientização.
* Executar avaliações periódicas de segurança.

---

# 📖 Glossário

| Termo          | Definição                                                                          |
| -------------- | ---------------------------------------------------------------------------------- |
| Evasão         | Técnicas utilizadas para reduzir a chance de detecção por mecanismos de segurança. |
| EDR            | Endpoint Detection and Response.                                                   |
| SIEM           | Security Information and Event Management.                                         |
| XDR            | Extended Detection and Response.                                                   |
| IOC            | Indicador de Comprometimento.                                                      |
| IOA            | Indicador de Ataque.                                                               |
| MITRE ATT&CK   | Framework que documenta táticas e técnicas observadas em ataques reais.            |
| Threat Hunting | Processo proativo de busca por ameaças em um ambiente.                             |
